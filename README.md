# 4.2 寸 300×400 全反射 SPI 模组（ST7305）资料与示例

**English：** [`README_EN.md`](README_EN.md)

---

> 本仓库提供该模组的 **ESP-IDF 示例工程**。数据手册与规格书待后续补充至 `docs/`。

## 产品概要

| 项目 | 说明 |
|:--|:--|
| 模组规格 | 4.2 英寸 **全反射式 LCD**（黑白），分辨率 **300×400**（横屏 400×300） |
| 接口 | **SPI** |
| 驱动芯片 | **ST7305** |
| 规格标识 | 产品资料中常用 **`4.2-lcd-300x400-spi-st7305`** 表示本规格 |
| 同尺寸其他规格 | 4.2″ **三色全反射** 见 **`4.2-lcd-300x400-spi-st7306`**（ST7306），驱动与色型不同、独立维护 |
